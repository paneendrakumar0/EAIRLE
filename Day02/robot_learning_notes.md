# Robot Learning Notes

- **What is reinforcement learning?**
  Reinforcement learning (RL) is a type of machine learning where an agent learns to make decisions by taking actions in an environment to maximize a cumulative reward. It learns through trial and error.

- **Why is RL useful for robots?**
  Robots operate in complex, unpredictable real-world environments. RL allows them to learn optimal behaviors directly from experience and sensor data, rather than relying on hardcoded, mathematically complex rules for every situation.

- **What is imitation learning?**
  Imitation learning is a technique where a model learns how to perform a task by observing a human or expert demonstration. The robot "imitates" the expert's behavior.

- **Why do robots need simulation?**
  Training robots in the real world is slow, expensive, and potentially dangerous (they could break or cause harm). Simulation provides a safe, fast, and scalable environment for robots to practice and learn millions of times before being deployed in the real world.

- **What is the difference between normal AI and embodied AI?**
  Normal AI (like ChatGPT or recommendation algorithms) operates purely in the digital domain. Embodied AI refers to an AI agent that has a physical body (or a simulated physical body) and can actively interact with and change its environment (e.g., a robot arm picking up an object).

# Robot Learning Journey

This repository tracks my two-year journey to becoming a **Robot Learning / Embodied AI Engineer**.

## Goal
To build a solid foundation in software engineering, machine learning, and robotics, culminating in the ability to design, train, and deploy intelligent agents capable of interacting with the physical world.

## 12-Volume Master Roadmap (2026-2028)

The curriculum is structured into 12 core volumes, covering everything from basic software engineering to advanced deep reinforcement learning and sim-to-real transfer:

- **Volume 1: Programming & CS Foundations** (Python, C++, DSA, Git, Linux)
- **Volume 2: Applied Mathematics for Robotics** (Linear Algebra, Calculus, Probability, Optimization)
- **Volume 3: Machine Learning & Deep Learning Fundamentals** (PyTorch, CNNs, Transformers)
- **Volume 4: Robotics Fundamentals & Kinematics** (Transformations, Forward/Inverse Kinematics, Dynamics)
- **Volume 5: Robot Operating System (ROS 2) & Systems Engineering** (Middleware, Nodes, CI/CD, Docker)
- **Volume 6: Computer Vision & Perception** (OpenCV, Object Detection, 3D Vision, Point Clouds)
- **Volume 7: State Estimation & SLAM** (Kalman Filters, Factor Graphs, Visual-Inertial Odometry)
- **Volume 8: Deep Reinforcement Learning** (MDPs, PPO, SAC, Imitation Learning)
- **Volume 9: Robot Simulation & Environments** (MuJoCo, Isaac Sim, URDF/MJCF formulation)
- **Volume 10: Planning & Control** (A*, RRT, Model Predictive Control, Whole-Body Control)
- **Volume 11: Robot Learning & Sim-to-Real** (Domain Randomization, RL Deployment, Hardware in the Loop)
- **Volume 12: Embodied AI & Advanced Topics** (Vision-Language-Action Models, End-to-End Control, Capstone Deployment)
